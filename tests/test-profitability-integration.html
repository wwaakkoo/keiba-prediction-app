🔍 収益性グラフ統合テスト

1メインアプリを開く

最初にメインアプリケーションを開いて、全てのJSライブラリを読み込んでください。

🏇 メインアプリを開く

2開発者ツール準備

メインアプリでF12を押して開発者ツールのコンソールを開いてください。

3システム存在確認

コンソールで以下のコマンドを実行して、システムの存在を確認してください：

// システム存在確認 console.log('ProfitabilityVisualizationSystem:', typeof ProfitabilityVisualizationSystem); console.log('showProfitabilityDashboard:', typeof showProfitabilityDashboard); console.log('Chart.js:', typeof Chart); console.log('RiskReturnAnalyzer:', typeof RiskReturnAnalyzer); console.log('InvestmentEfficiencyCalculator:', typeof InvestmentEfficiencyCalculator);

4グラフダッシュボード表示テスト

コンソールで以下のコマンドを実行して、収益性ダッシュボードを表示してください：

// ダッシュボード表示 showProfitabilityDashboard(); // または直接表示 showProfitabilityDashboardDirect();

5HTML要素確認

ダッシュボード表示後、HTML要素が生成されているか確認：

// HTML要素確認 console.log('profitabilityDashboard:', document.getElementById('profitabilityDashboard')); console.log('riskReturnChart:', document.getElementById('riskReturnChart')); console.log('efficiencyChart:', document.getElementById('efficiencyChart')); console.log('learningGraphsSection:', document.getElementById('learningGraphsSection'));

6データ取得テスト

データ取得関数の動作確認：

// データ取得テスト console.log('リスクリターンデータ:', ProfitabilityVisualizationSystem.getRiskReturnData()); console.log('投資効率データ:', ProfitabilityVisualizationSystem.getInvestmentEfficiencyData()); console.log('収益性データ:', ProfitabilityVisualizationSystem.getProfitabilityChartData());

7チャート作成テスト

個別チャート作成をテスト：

// チャート個別作成テスト try { ProfitabilityVisualizationSystem.createRiskReturnScatterChart(); console.log('✅ リスクリターンチャート作成成功'); } catch (error) { console.error('❌ リスクリターンチャート作成失敗:', error); } try { ProfitabilityVisualizationSystem.createEfficiencyScatterChart(); console.log('✅ 投資効率チャート作成成功'); } catch (error) { console.error('❌ 投資効率チャート作成失敗:', error); }

8手動初期化テスト

システムを手動で初期化してテスト：

// 手動初期化 try { ProfitabilityVisualizationSystem.initialize(); console.log('✅ システム初期化成功'); } catch (error) { console.error('❌ システム初期化失敗:', error); } // 全チャート更新 try { ProfitabilityVisualizationSystem.refreshAllCharts(); console.log('✅ 全チャート更新成功'); } catch (error) { console.error('❌ 全チャート更新失敗:', error); }

9結果確認

各ステップの実行結果を記録して、以下の情報を確認してください：

  • どのシステムが読み込まれているか
  • HTML要素が正しく生成されているか
  • データが正しく取得できているか
  • エラーメッセージの内容

📝 期待される結果

正常な場合、以下が確認できるはずです：

  • ✅ 全システムが 'function' または 'object' として定義
  • ✅ HTML要素が存在（null でない）
  • ✅ データ取得関数が配列/オブジェクトを返す
  • ✅ チャート作成がエラーなく完了

⚠️ 問題発生時の対処

エラーが発生した場合：

  1. エラーメッセージをコピーして報告
  2. どのステップで失敗したかを特定
  3. ブラウザの更新（Ctrl+F5）を試す
  4. Chart.jsの読み込み状況を確認
